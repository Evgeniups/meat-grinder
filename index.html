Мясорубка электрическая - ваш
незаменимый помощник!

Электрическая мясорубка предназначена для изготовления фарша из мяса, рыбы, грибов, нарезки овощей и фруктов, профилирования песочного и пельменного теста. Тип мясорубки — шнековый.

Данная модель снабжена функцией "реверс" для предотвращения уплотнения продуктов внутри мясорубки предусмотрен режим обратного вращения шнека. Корпус мясорубки изготовлен из прочного пластика, на нижней части корпуса расположены прорезиненные ножки для дополнительной устойчивости.

Мощность двигателя 2000Вт. 2 острых металических измельчителя из нержавеющей стали. Головка корпуса из алюминия. Набор решёток для приготовления фарша. Низкий уровень шума. Крестообразный измельчитель из нержавеющей стали.

Электрическая Мясорубка – это стильный дизайн, высокая степень надежности, большое количество функциональных возможностей, которые делают данную модель незаменимой помощницей и идеальным дополнением любой кухни. Теперь Вы можете готовить с удовольствием и радовать кулинарными шедеврами свою семью.
